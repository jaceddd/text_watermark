Add invisible characters on copy to protect content.

This section makes it so when somebody tries to copy something from your site to paste elsewhere, invisible spaces are added to the end. Try copy and pasting any text on this page! The message should be unique and uses the standard non encrypted version to hide the message. To make sure this has worked, you can paste the text into this section to decode the hidden message contents. or this section to visualize

If you want to add this to your website, you need to edit copy_and_paste_adder.js and include it in your page. Look in the code for the line:
var message = 'ADD YOUR INVISIBLE MESSAGE HERE';
